# Fake Secrets Demo Repository

This repository contains **intentionally fake secrets** designed for demonstration, testing, and educational purposes.

These values **do not provide access to any real services** and **are not connected to real accounts**. They exist purely to simulate common secret formats used in applications.

## Purpose

This project is useful for:

* Demonstrating **secret scanning tools**
* Testing **security detection systems**
* Teaching developers about **secret exposure risks**
* Running **security workshops or demos**
* Experimenting with **CI/CD secret detection pipelines**

## Important Notice

All credentials in this repository are **100% fake**.

Examples include simulated:

* API keys
* AWS credentials
* Private keys
* Tokens
* Database passwords
* OAuth secrets

None of these values are valid or usable.

If you are looking for real credentials, you will **not find them here**.

## Use Cases

You can use this repository to:

* Test tools like:

  * GitHub Secret Scanning
  * TruffleHog
  * GitLeaks
  * detect-secrets
* Demonstrate **how secrets can accidentally leak into repositories**
* Practice **secure development workflows**

## Do Not Use for Production

This repository is strictly for:

* Learning
* Demonstrations
* Security testing

Do **not** use these values in real applications.
